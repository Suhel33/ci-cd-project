# 🚀 Royal Events - Automated CI/CD Deployment on AWS

This project demonstrates a fully automated CI/CD pipeline implemented using GitHub Actions.
The pipeline automatically deploys a static website to an AWS EC2 server running Nginx whenever code is pushed to the main branch

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Cloud:** AWS (EC2 Instance)
- **Web Server:** Nginx
- **DevOps:** GitHub Actions (CI/CD)
- **Automation:** SSH Action for remote deployment
- **Authentication:** SSH (Key-based)
- **Operating System:** Ubuntu Linux

## ⚙️ Key Features
- **Automated Deployment:** Any change pushed to the `main` branch is automatically deployed to the AWS server.
- **Zero Manual Intervention:** No need to manually SSH or move files; GitHub Actions handles everything.
- **Permission Management:** Secure file handling with Nginx permissions.
